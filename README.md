# ngrok-helper

Desenvolvido em 2020, durante a pandemia, um software que automatizou o uso do Ngrok para abertura de portas TCP, eliminando a necessidade de configuração manual de roteadores e NAT. A ferramenta permitia criar e gerenciar túneis TCP de forma simples, viabilizando jogos online de Minecraft entre amigos, mesmo em redes domésticas restritivas.
